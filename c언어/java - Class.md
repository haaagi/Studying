# java - Class

#### 객체 지향 프로그래밍 ( Object Oriented Programming ) - OOP

- 객체를 기반으로 하는 프로그래밍
- 객체를 정의 하고, 객체의 기능을 구현하며, 객체간의 협력을 구현

#### 절차 지향 프로그래밍 ( Procedural Programming )

- 시간이나 사건의 흐름에 따른 구현
-  C언어



#### 규칙

- 한 파일 안에 `public` 클래스는 1개 여야 한다.
- `public` 이 아닌 class 는 생성 가능
- `public` 클래스 이름은 java 파일 이름과 같아야 한다.
